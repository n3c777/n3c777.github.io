Here are my links to demonstration videos of some of my projects

  • Decilumen Demo

    • An application that synchronizes audio and lights. Integrates multiple technologies, including React, Python, and C++, running in a Docker container and an ESP32.

  • Lose The Leftovers Demo

    • A fully functional capstone mobile app developed in a group for a client. Built with React Native (TypeScript) and Django, the app allows users to donate unused food to those in need.

  • Mock IoT Gateway

    • An application built in C, running on a Linux machine. It uses TCP/Posix sockets to securely receive and send data, with sensor data mocked by a Python script

  • Antipode-Find

    • A project that lets users find the antipodal point of any location on Earth and identify the country it falls in. The map and markers are displayed using Leaflet.js, while Flask powers the backend. GeoPandas runs a point-in-polygon algorithm to determine the antipodal country's location using a local GeoJSON file of the national borders.
